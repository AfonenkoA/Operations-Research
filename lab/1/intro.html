Пример 24.1 Чтобы попасть из пункта А (остановка автобуса) в пункт В (лодочная станция) человек должен пройти сначала по асфальтовой дороге шириной \(S\), а затем по песчаному пляжу шириной \(s\). Скорость передвижения по асфальту \(V\) , скорость передвижения по песку \(v\). Спрашивается, в каком месте нужно свернуть с асфальтовой дороги, чтобы затратить меньше времени на путь.

Решение 24.1. Решим задачу с конкретными значениями, в виду сложности аналитического решения (сводится к уравнению 4 степени): \(D = 100\), \(S = 40\), \(V = 5\), \(s = 60\), \(v = 3\)

Критерий оптимальности задается функцией \(t(x)\), которую надо минимизировать, изменяя варьируемый параметр \(х\). Остальные параметры задачи \((v,V,D,s,S)\) являются фиксированными. \[t(x) = \frac{\sqrt{x^2+S^2}}{V} + \frac{\sqrt{(D-x)^2+s^2}}{v}\] \[\begin{cases} \min{t(x)} - ?\\ \frac{dt}{dx}=0\\ \frac{d^2t}{dx^2}>0 \end{cases}\,\] \[t'(x)=\frac{x}{V \sqrt{S^2+x^2}}-\frac{d-x}{v \sqrt{(d-x)^2+s^2}}\]
