Приложение L — Nloptr. Локальная. Ограничения равенства

Пример L.1 Рассмотрим на основе примера Глава 15.3

f <- function(par)
{
  p1 <- par[[1]]
  p2 <- par[[2]]
  p1^2 + p2^2 + p1*p2 + 2*p1+4*p2 + 5
}

Также для решателю могут быть переданы градиент

f_g <- function(par)
{
  p1 <- par[[1]]
  p2 <- par[[2]]
  c(2*p1 + p2 + 2, 2*p2 + p1 + 4)
}

Теперь добавим для функций обёртку. Задание прямоугольных ограничений неравенства рассматривается в пункте

par <- ParamHelpers::makeNumericParamSet(
    len = 2L,
    lower = c(-5, -5),
    upper = c(5, 5))

objfn <- smoof::makeSingleObjectiveFunction(
  fn = f,
  par.set = par)

objfn_g <- smoof::makeMultiObjectiveFunction(
  fn = f_g,
  par.set = par,
  n.objectives = 2)

Стартовая точка для всех методов. Для использования метода необходимо выбрать начальное приближение

x0 <- c(5,2)

Задаваемые ограничения дожны быть преведены к форме

\[ \begin{cases} g(x) \leq 0 \\ h(x) = 0 \\ \end{cases} \]

Преобразуем ограничение из примера

$$ p_1^2 - p_1 p_2 = -2

$$

к форме требуемой в пакете

\[ p_1^2 - p_1 p_2 + 2=0 \]

Определим функцию ограничения равенства

constr_eq <- function(par)
{
  p1 <- par[[1]]
  p2 <- par[[2]]
  p1^2 - p1*p2 +2
}

И якобиан

constr_eq_jac <- function(par)
{
  p1 <- par[[1]]
  p2 <- par[[2]]
  c(2*p1-p2, -p1)
}

И их обёртки

constr_eq_fn <- smoof::makeMultiObjectiveFunction(
  fn = constr_eq,
  par.set = par,
  n.objectives = 2)

constr_eq_jac_fn <- smoof::makeMultiObjectiveFunction(
  fn = constr_eq_jac,
  par.set = par,
  n.objectives = 2)

  x0 = x0,
  eval_f = f,
  eval_grad_f = f_g,
  eval_g_eq = constr_eq,
  eval_jac_g_eq = constr_eq_jac,
  opts = list(
    algorithm = 'NLOPT_LD_SLSQP',
    xtol_rel = 1.0e-8
  )
)
